WIKIVOLLEY
Portal web profesional dedicado a la actualidad y gestión de datos del mundo del voleibol. El proyecto consiste en una plataforma dinámica que integra noticias, ofertas de empleo y formación especializada.

Tecnologías Utilizadas
Frontend
HTML5: Estructura semántica del contenido.

CSS3: Diseño personalizado con enfoque en la experiencia de usuario.

Responsive Design: Adaptación completa para dispositivos móviles, incluyendo correcciones específicas para renderizado en motores WebKit (iOS) y Blink (Android).

Backend
PHP: Procesamiento de datos en el servidor y generación de contenido dinámico.

MySQL: Gestión de base de datos relacional para el almacenamiento de noticias, ofertas laborales y cursos.

Características Técnicas
Arquitectura Cliente-Servidor: Conexión eficiente mediante PHP para la recuperación de registros en tiempo real.

Compatibilidad Multiplataforma: Implementación de Media Queries avanzadas para garantizar la nitidez de las imágenes y la legibilidad de la tipografía en diferentes sistemas operativos móviles.

Interfaz Dinámica: Sistema de tarjetas (grids) que se reajusta automáticamente según la resolución de pantalla.

Instrucciones de Instalación
Clonar el repositorio.

Importar el archivo SQL adjunto en su gestor de bases de datos MySQL.

Configurar las credenciales de acceso en el archivo conexion.php.

Alojar en un servidor con soporte para PHP 7.4 o superior.
